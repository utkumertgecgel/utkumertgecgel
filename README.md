## 🚀 Featured Projects

<details>
<summary><b>🌐 Gesture Globe</b> &nbsp;—&nbsp; <em>Computer Vision · Real-Time Gesture Control · Interactive 3D Visualization</em></summary>

<br/>

> A real-time computer vision system that translates hand gestures into interactive globe navigation. Combines MediaPipe's hand landmark detection pipeline with a 3D rendering engine to deliver a touchless, gesture-driven geospatial interface — eliminating the need for physical input devices entirely.

<br/>

| Attribute | Details |
|:---|:---|
| **Stack** | Python · OpenCV · MediaPipe · Three.js · FastAPI · WebSocket |
| **Scale** | Real-time 30fps gesture recognition · Sub-50ms end-to-end latency |
| **Performance** | Optimized landmark detection with custom gesture classification layer |
| **Architecture** | Decoupled Python inference backend with WebSocket-streamed frontend renderer |
| **Domain** | Computer Vision · Human-Computer Interaction · Geospatial Visualization |
| **Repository** | [![GitHub](https://img.shields.io/badge/GitHub-gesture--globe-7C3AED?style=flat-square&logo=github&logoColor=white)](https://github.com/utkumertgecgel/gesture-globe) |

**Engineering Highlights:**
- Built a custom gesture classifier on top of MediaPipe's 21-point hand landmark model, supporting pan, zoom, rotate, and select gesture vocabulary
- Designed a low-latency WebSocket communication layer between the Python inference engine and the browser-based 3D renderer
- Achieved stable real-time performance through frame-level pipeline optimization and async frame processing
- Architected the system with a clean separation between the CV backend and visualization frontend, enabling independent scaling and testing

<br/>

</details>
